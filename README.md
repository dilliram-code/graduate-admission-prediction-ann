# Graduate Admission Prediction using ANN: Mini Project😅

This project predicts the chances of graduate admission using an Artificial Neural Network (ANN). The model is built with TensorFlow and Keras and trained on academic and profile-related features such as GRE score, TOEFL score, CGPA, SOP, LOR, and research experience.

The project includes:
- Data preprocessing and feature scaling
- ANN model building and training
- Model evaluation and prediction
- Data visualization using Matplotlib

## Technologies Used
- Python
- TensorFlow
- Keras
- Scikit-learn
- Matplotlib
- Pandas
- NumPy

## Goal
The main goal of this project is to apply deep learning techniques to predict admission probabilities based on student academic profiles.
